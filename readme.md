# 🧠 Mind Trials — AI Cognitive Game Platform

## 🚀 Overview
Mind Trials is an interactive AI-powered game that evaluates human cognitive behavior through multiple trials:
- Logic reasoning
- Memory recall
- Psychological analysis

## 🎮 Features
- Multi-level gameplay
- AI-driven interaction
- Cognitive profiling system
- Leaderboard system
- Real-time metrics
- Report generation

## 🧠 How It Works
The AI observes:
- Decision speed
- Consistency
- Memory accuracy
- Psychological responses

Then generates a behavioral profile.

## 🛠 Tech Stack
- Python
- Streamlit
- OpenAI API (optional)
